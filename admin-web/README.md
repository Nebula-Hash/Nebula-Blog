# 博客后台管理系统

## 技术栈
- Vue 3
- Vite
- Naive UI
- Pinia
- Vue Router
- Axios

## 安装依赖

```bash
cd admin-web
npm install
```

## 启动开发服务器

```bash
npm run dev
```

访问地址: http://localhost:3001
