## 不再作为首轮阻塞项的内容

以下内容可以做，但不应阻塞首次生产部署：

- 团队培训
- 系统化告警平台
- SQL 注入 / XSS 专项安全测试
- CDN 接入
- 多副本扩容
- Kubernetes 迁移准备
